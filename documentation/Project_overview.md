
# project overview: Maternal and Child health County Analytics

# Background
Maternal and child health remains a key public health priority in Kenya.
County-level analysis is essential for identifying disparities and guiding interventions.

# Objectives
Analyze maternal and child healt indicators at county level
Identify geographic patterns using visualization
Demostrate healthcare analytics skills end-to-end

# Key Questions Answered
Which counties have higher mortality rates
How does ANC coverage vary geographically?
Where are high risk regions located?

# Intended Users
Public health officers 
County-health teams
Healthcare Analysts
Recruiters and clients

# Data Description
Synthetic county-level MCH Indicators were generated to reflect realistic Kenyan health reporting patterns while maintaining data privacy.

# Outputs
Clean CSV datasets
Interactive Power BI dashboard
Professional documentation

# Limitations
Synthetic data
Simplified county modeling
